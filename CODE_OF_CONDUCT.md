# Code of Conduct

This project is meant to be useful, focused, and safe for developers sharing
local usage-tooling feedback.

## Expected Behavior

- Be respectful and assume good intent.
- Keep issues and PRs focused on the project.
- Share only redacted diagnostics when discussing Codex usage data.
- Respect that local paths, thread titles, prompts, responses, and logs can be
  private.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Posting private prompts, raw Codex logs, API keys, tokens, or other secrets.
- Pressuring maintainers or contributors to disclose private local usage data.
- Repeated off-topic comments after a maintainer asks to refocus.

## Enforcement

Maintainers may edit or remove comments, close issues, or block participants to
keep the project safe and productive. Security-sensitive reports should follow
the process in [SECURITY.md](SECURITY.md).
